# ワイルドカード

コマンド実行時に複数のファイル指定を行うための特殊記号。

- `*`: 空文字を含む任意の文字列
- `?`: 任意の1文字
- `[...]`: `[]` 内に含まれる任意の1文字
- `[!...]`: `[]` 内に含まれない任意の1文字
- `{A,B,C}`: A, B, C のいずれか

## ワイルドカードとドットファイル

先頭に `.` がくるファイルはワイルドカードで例外処理されるようになっている。

```bash
cp * some_dir/
```

とすると、先頭が `.` ではない全てのファイルがコピーされる。

```bash
cp .* some_dir/
```

とすると、「先頭が `.` である全てのファイル」がコピーされる。
