# kill コマンド

kill コマンドは、指定したシグナルをプロセスまたはプロセスグループに送信するためのコマンドです。主にプロセスの終了や制御に使用されますが、他のシグナルも送信できます。

```bash
kill [オプション] <PID>...
```

## シグナルとは

シグナルは、プロセスに対して特定のアクションを指示するための通知メカニズムです。
よく使うシグナルには以下のようなものがあります。

| 種類      | 値  | 説明                                                                     |
| --------- | --- | ------------------------------------------------------------------------ |
| `SIGINT`  | 2   | プロセスに割り込みを送信します（通常は `Ctrl+C` によって送信されます）。 |
| `SIGKILL` | 9   | プロセスを強制終了します。                                               |
| `SIGTERM` | 15  | プロセスに終了を要求します。デフォルトのシグナルです。                   |

全部のシグナルを確認するには、`kill -l` コマンドを使用します。

※ カーネルによってサポートされているシグナルは異なる場合があります。

```bash
kill -l
```

## 例

このようにシグナルは、シグナル名またはシグナル番号で指定できます。

```bash
kill -SIGINT 1234    # PID 1234 のプロセスに SIGINT シグナルを送信
kill -9 1234         # PID 1234 のプロセスを強制終了 (SIGKILL)
kill 1234            # PID 1234 のプロセスにデフォルトの SIGTERM シグナルを送信
```
