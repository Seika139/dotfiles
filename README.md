# dotfiles

## install

```bash
source install.sh
```

## uninstall

```bash
source uninstall.sh
```

## dotfiles とは

ホームディレクトリに置いてあるドット(.)から始まる設定ファイル(.bashrc とか)を管理しているリポジトリのこと。
先輩につくることを勧められたので私も作ってみます。
